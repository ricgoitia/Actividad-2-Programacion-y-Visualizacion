# Actividad 2 - Programación y Visualización

## Objetivo
Analizar la base de datos del Sistema Nacional de Investigadoras e Investigadores (SNII) 2024 mediante Python para identificar patrones relevantes utilizando técnicas de análisis y visualización de datos.

## Herramientas utilizadas
- Python
- Pandas
- Matplotlib
- Google Colab

## Análisis realizados
- Cantidad de investigadores por nivel de distinción.
- Monto promedio anual por nivel.
- Distribución del monto anual.
- Boxplot de montos por nivel.
- Distribución geográfica de investigadores.
- Monto promedio por área de conocimiento.
